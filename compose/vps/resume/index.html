The forge

hart forge is my homelab - a self-hosted platform I run end to end, managed as code. everything below runs on hardware I own plus one small VPS, with configs versioned in my forgejo. it's also where the day-job skills get sharpened: identity, networking, automation, monitoring - same problems, my hardware.

Virtualization Proxmox VE running a mix of VMs and LXC containers
Config as code Ansible playbooks for fleet health checks, updates, and backups
Containers Docker Compose per host, pinned images, automated update watching
Edge & security Caddy reverse proxy with CrowdSec IPS, Cloudflare DNS
Identity passkey-based OIDC SSO with Pocket ID and forward auth
Self-hosted git Forgejo primary on the VPS with a LAN mirror
Media & photos Jellyfin with a fully automated pipeline, Immich for photos
Monitoring Uptime Kuma, Beszel metrics, and ntfy push alerting
Agentic ops a persistent Claude-based agent works the fleet with me - ops, deploys, and this site
